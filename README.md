# ü§ñ [Critical Thinker](https://github.com/AlvesMH/Critical-Thinker)  |  [Live-Demo](https://critical-thinker.onrender.com/)
## Multi-Agent Critical Thinking Analysis (Paul & Elder Framework)
**Critical Thinker** is a lightweight web application that supports rigorous evaluation of arguments, essays, policy proposals, and academic texts using **Paul & Elder‚Äôs Critical Thinking Framework**. The application operationalizes the framework through four independent analytical perspectives (‚Äúagents‚Äù) and produces a consolidated report that can be downloaded as a professionally formatted PDF.

The design principle is intentionally pedagogical and method-driven: the app is not a replacement for critical thinking, but a structured aid to **surface assumptions, scrutinize evidence, evaluate reasoning quality, and identify implications**‚Äîall while maintaining explicit standards of intellectual rigor.
